Ego docker store is a git registry where you can find and install various applications designed to enhance the functionality of your Ego robot. These applications can range from simple utilities to complex systems that enable advanced behaviors and interactions.
# How to Access the Ego Docker Store
![-](/img/docker-guide.png)


From the IIT-SoftBots repository, you can navigate to the packages section. There, you will find all the available applications for the Ego robot. Each application comes with a detailed description, installation instructions, and user reviews to help you make informed decisions.



## Popular Dockers
Here are some popular docker applications available in the Ego docker store:
- **Alterego conversation**: A voice-controlled assistant that helps you manage tasks and control your Ego robot using natural language commands.
- **Alterego object recognition**: an application that enables your Ego robot to recognize objects and infer their 3D pose.
- **Alterego face identification**: An application that allows your Ego robot to identify and remember faces, enhancing its ability to interact with users. In addition to identifying faces, this application also enables the Ego robot to recognize human emotions, infer gender, age and ethnicity, allowing for more empathetic interactions.

## How to contribute
If you have developed an application that you believe would be beneficial to the Ego robot community, you can contribute it to the Ego docker store.
